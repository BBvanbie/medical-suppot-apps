## 保管ルール

- repo 内の本文書は `様式の正本` とし、実名版や実記録は導入組織側の管理場所へ保存する。
- 主保管場所は `導入組織の文書管理 / 共有フォルダ / チケット / GRC 台帳` のいずれか 1 つへ固定する。
- 副保管場所は、主保管場所が参照不能になった場合でも監査時に取り出せる場所を指定する。
- 閲覧権限は `管理責任者 / 企画管理者 / システム運用担当 / 監査担当` の必要最小限に絞る。
- 緊急連絡先は紙またはオフライン参照手段を 1 系統以上持つ。

## 更新ルール

- 人事異動、委託先変更、窓口変更があった場合は即日更新する
- 少なくとも月次点検時に内容を再確認する
- 実名版は repo にコミットせず、導入組織側の管理場所に保管する
- 保管場所を変更した場合は、旧保管場所からの削除 / 失効確認まで同日中に行う
- 監査時に 5 分以内で提示できない保管場所は `未固定` とみなし、是正対象にする
