# オフライン対象データ一覧

最終更新: 2026-04-01

## 保存対象

| 種別 | 内容 | 用途 |
| --- | --- | --- |
| 事案フォーム下書き | case payload, dispatch context, tab 入力内容 | EMS 編集継続、復元 |
| 未送信キュー | 受入要請送信、相談返信などの pending payload | 後続再送 |
| 病院検索キャッシュ | 病院一覧の簡易検索用 row | オフライン検索 |
| 同期状態 | 最終同期時刻、再送状態 | settings / sync 表示 |

## 制限

| 操作 | オフライン対応 | 備考 |
| --- | --- | --- |
| 事案フォーム編集 | 可 | ローカル下書き中心 |
| 病院検索 | 簡易検索のみ | キャッシュ依存 |
| 受入要請送信 | キュー化 | 即時送信しない |
| 相談返信 | キュー化 | 即時送信しない |
| 搬送決定 / 搬送辞退 | 不可 | online 必須 |

## 競合方針

- `手動解決`
- `server優先`
- `フォーム再保存`

完全自動マージや差分比較 UI は未対応です。
