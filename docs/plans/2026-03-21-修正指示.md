# A側事案入力画面修正指示
これはA側の事案情報の概要_バイタルタブでの入力内容の修正指示です。
- 痛み系は発症時間（厳密ではない）、経過（改善傾向、増悪傾向、変わらずなど）を入れる
- 痙攣は部位、性状、継続時間を入れる
    - 部位
        - 全身
        - 局所
            - 局所なら部位を選択
                - 部位
                    - 右上肢
                    - 右下肢
                    - 左下肢 
                    - 右下肢
                    - 顔面
                    - 上半身
                    - 下半身など（推奨を追加可能）
                    - その他（テキストinputを出す）
    - 性状
        - 強直性
        - 間代性
    - 経過時間
        - number
        - 自動で":"を入力
        - 患者サマリーでは"約～分"と表示
- 動悸に発症時行動を追加
    - 安静時
    - 運動時
    - 労作時など 
- 全てにおいてその他を選んだらテキストinputを表示
- 喀痰の性状リストに血性があるのに、別で血性＋－確認困難があり重複している
- 吐下血は吐血と下血に分離させる
- 痛み関連の部位は複数選択可能とする
- 意識障害は＋を選択したら普段のレベル選択枠を出す
- 顔面麻痺は＋なら部位を選択
    - 右眼瞼
    - 左眼瞼
    - 右口角
    - 左口角など
