# DISPATCH指令隊フィルタ実装

日付: 2026-04-27

## 目的

DISPATCHの指令起票で、出場隊が増えた状況でも目的の隊を素早く選べるようにする。

## 変更内容

- `listDispatchTeamOptions()` の戻り値に `division` を含める。
- 指令起票フォームの出場隊選択に `方面` フィルタを追加する。
- 指令起票フォームの出場隊選択に `隊検索` を追加し、隊名・隊コードで絞り込めるようにする。
- 表示中の隊だけを一括選択 / 一括解除できる補助操作を追加する。
- 絞り込み中でも、既に選択した隊は選択状態を保持する。
- 条件に一致する隊がない場合は空状態メッセージを表示する。

## 検証方針

- `npm run check` 通過。
- `npx.cmd playwright test e2e/tests/dispatch-flows.spec.ts --reporter=line` 通過。dispatch E2Eは `3 passed`。
- dispatch E2Eでは、1方面で `E2E 本部機動第1` を選び、2方面へ切り替えて `E2E 本部機動第2` を検索・選択し、2隊分の起票が成立することを確認する。
