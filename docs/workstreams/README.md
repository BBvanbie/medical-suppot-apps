# workstreams

このディレクトリは、現在進行中または継続管理が必要なテーマごとの要約を置く。
役割は「残作業と再開地点の整理」であり、詳細な判断理由は `docs/plans/` を参照する。

## 使い分け

- 現在の再開点: [current-work.md](/C:/practice/medical-support-apps/docs/current-work.md)
- テーマ別の状態整理: `docs/workstreams/*.md`
- 履歴と詳細判断: `docs/plans/*.md`
