# presentation assets

外部説明用 HTML スライドで使う画像素材を置きます。

## 推奨ファイル名

- `ems-case-input.png`
- `ems-hospital-search.png`
- `ems-send-history.png`
- `hospital-requests.png`
- `hospital-consult.png`
- `admin-home.png`
- `admin-monitoring.png`
- `admin-devices.png`
- `offline-queue.png`
- `training-mode.png`

## 画像ルール

- TRAINING / demo data だけを使う。
- 実名、実患者情報、実住所、実電話番号を含めない。
- 必要なら画像編集でマスクする。
- 16:9 または 4:3 に近い横長画像を優先する。
- 画像が未取得の場合でも、HTML 側は placeholder frame で成立させる。
