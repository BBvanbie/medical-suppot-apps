# presentations

外部説明用の HTML スライドと関連素材を置くディレクトリです。

## 現在の資料

- `emergency-transport-support-system.html`
  - 「救急搬送支援システム」外部説明用の単一 HTML スライド
  - 対象: 病院、消防、自治体、経営層、行政担当
  - 目的: PoC 承認、導入検討、予算 / 協力体制確保

## 運用ルール

- 実名、実患者情報、実電話番号、実住所は入れない。
- 導入先名は `A消防本部`、`B総合病院`、`C市医療圏` のような仮名で扱う。
- スクリーンショットは TRAINING / demo data のみを使う。
- 外部共有前に画像内の個人名、電話番号、住所、病院名を確認する。
- HTML はブラウザで直接開ける単一ファイルを基本にする。
